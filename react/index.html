React Projects

  • Emotehunt

    Emotehunt allows users to search for emojis and copy them to the clipboard with just a click.

  • NoteWorthy

    NoteWorthy is a note-taking web application offering rich text formatting, and download functionality. Users can effortlessly create, format their notes with features such as headings, inline code, and customizable styling options, making it an ideal tool for effective note management.

  • ImageQuest

    ImageQuest web application designed to streamline the process of finding and displaying images based on user input. It offers a seamless experience for users to search for images by entering specific keywords or choosing from a set of predefined options.

  • Shopkart

    Shopkart is a designed to provide users with a seamless shopping experience. The application allows users to browse through a variety of products, search for specific items, filter products by categories, add items to their cart, and proceed to checkout.

  • Winfolio

    Winfolio is a web application designed for individuals to record and showcase their accomplishments, enabling users to log and share their achievements.
