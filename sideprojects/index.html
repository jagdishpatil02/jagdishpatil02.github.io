Side Projects

  • Social Media Blocker (Javascript)

    Social media blocker is a chrome extension which will not allow you to use social media websites while working with warning. This extension will help you to improve productivity.

  • Drink Water (Javascript)

    Drink Water (Jal Lijiye) is a chrome extension that will remind you drink water after every 20 mins. Drink Water (Jal Lijiye) is an extension which will help you to stay hydrated.

  • Hotdog or not (Angular, Hugging Face)

    Hotdog or not is a web application designed to check if the uploaded image is a hot dog or not using LLM model on Hugging Face

  • Accessible Code Generator (Angular, OpenAI LLM)

    The Accessible Code Generator is a GPT wrapper designed to assist developers in suggesting accessible code changes based on WCAG (Web Content Accessibility Guidelines) standards. This tool utilizes the gpt-3.5-turbo model developed by OpenAI.

  • Winfolio (React, Supabase)

    Winfolio is a web application designed for individuals to record and showcase their accomplishments, enabling users to log and share their achievements.

  • Diwali Wish generator (Next.js)

    Diwali wish generator enables users to create and share diwali wishes.

  • Bluesky Post Sentiment Analyzer (Next.js, Hugging Face)

    Bluesky Post Sentiment Analyzer which scores posts based on positivity, negativity, and overall tone using LLM model on Hugging Face! It's a quick way to get insights into the vibe of any post on bluesky.

  • Bluesky Pass Generator (Next.js)
